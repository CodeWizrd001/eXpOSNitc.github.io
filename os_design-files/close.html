Close System Call



Arguments: fileDescriptor (Integer)

Return Value:

0 Success

Description: The Close operation takes as input the fileDescriptor of the file to be closed. It closes one open instance of the file by making corres. Otherwise, it deleteponding updations in File Table and Per Process Resource Table.

Algorithm:

If fileDescriptor is invalid

then Return -1

If Per Process Resource Table entry corresponding to fileDescriptor is invalid

then Return -1

Store the File Table Index corresponding to that entry in FTIndex

Update File Table

1. Find the File Table Entry corresponding to FTIndex

2. Decrement the File Open Count. If the File Open Count becomes 0, invalidate the File Table entry.

Invalidate the Per Process Resource Table entry corresponding to fileDescriptor

Return from System Call with 0, indicating success
