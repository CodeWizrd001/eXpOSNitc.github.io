Create System Call



Arguments: Filename (String)

Return Value:

0 Success
-1 No space for file

Description: The Create operation takes as input a filename and creates an empty file by that name. If a root entry for the file already exists, then the system call returns 0 (success). Otherwise, it creates a root entry for the file name, sets the file type to DATA and file size to 0. Note that the file name must be a character string and must not be “root”.

Algorithm:

If file is present in Inode Table

then Return 0

If no free entry in Inode Table

then Return -1

Store index of free entry in InodeIndex

Note: You can optimize the above two searches by storing the first free entry in InodeIndex while searching for file with the given filename.

In the Inode Table entry corresponding to InodeIndex , set file name to Filename, file size to 0 and file type to “DATA ”.

The entry for the file in the root is at InodeIndex. Set the filename field with Filename, file size to 0 and file type as “DATA”. Increment the file size of root.
