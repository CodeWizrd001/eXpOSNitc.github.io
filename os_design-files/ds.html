Datastructures

eXpOS uses various data structures to maintain information about files, processes, and semaphores along with memory pages and disk blocks.There are two kinds of data structures which include: Disk Data Structures and Memory Data Structures.

Disk Data Structures

Those data structure which will be stored in the disk and will be retained after machine shuts down. A memory copy of these structures are maintained by XOS when it is run.

  • Disk Free List

  • Inode Table

Memory Data Structures

Those data structures which are stored in the main memory of the machine and are required only when the operating system is running. They are lost when the system shuts down

  • Process Table

  • Global File Table

  • Global Semaphore Table

  • System Status Table
