Shutdown System Call



Arguments: None

Return Value:None

Description: Shutdown system call terminates all processes and halts the machine.


Algorithm:

while disk is not free do     /* Check status field in Disk Status Table */

Change the state of the process to WAIT_DISK and call the scheduler.

endwhile

/* The following store operations are not done asynchronously */

Store Inode Table to the disk.

Store dirty pages contained in buffer to the disk.     /* Check dirty bit in Buffer Table */

Store Disk Free List to the disk.

Halt the machine.




Shell System Call



Arguments: None

Return Value:None

Description: It is created by the init process to act like a console. It accepts a filename, forks itself and execs the file given as filename. These steps are repeated to accept and execute other files from the console.


Algorithm:

while 1 do

Get the program to be executed using read system call

childPID = fork();

If (childPID == 0)   /*Only child process will do the exec */

Execute the program given by read.

else Wait for child to finish execution.     /* Using Wait system Call*/

endwhile




Idle System Call



Description: It is a user program which is created and executed by the bootstrap loader. It is stored in the disk and will be loaded to memory by the bootstrap loader. The main purpose of the idle process is to run as a background process in an infinite loop. The page table and process table for the init process will be set up by the bootstrap loader. The PID of the init process is fixed to be 0.


Algorithm:

while 1 do

endwhile
