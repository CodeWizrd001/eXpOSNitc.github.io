Steps to Load/Store



In the Disk Status Table, do the following steps

1. Set the Load/Store bit to 0 (load) or 1 (store).

2. Set the Page Number field to the number of the page being stored / to which the disk block is loaded.

3. Set the Block Number field to the number of the block to which the page is stored / is being loaded.

4. Set the PID to the PID of the process invoking the disk-memory transfer. If the load/store instruction was invoked by scheduler, set the PID to -(pid) where pid is the identifier of the process for which load/store operation was initiated.

Issue the load/store machine instruction.
