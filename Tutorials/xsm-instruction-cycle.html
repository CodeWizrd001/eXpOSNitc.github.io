Understanding XSM Architecture



In this tutorial we will be walking you through the instruction execution cycle of XSM architecture.
For this purpose, we will be using the following SPL program that will display the contents of the first entry of the root file. You can refer the SPL specification here.

The spl program is given below: 

//Program to print the first entry of root file

alias counter R0;
alias word R1;
counter = 0;

load(62,5);

while(counter < 8) do
  word = [62 * 512 + counter];
  print(word);
  counter = counter + 1;
endwhile;


This corresponds to the following XSM code. 

// Initialising counter(R0) to 0
MOV R0, 0	

// Load block num 5 (root file) to page 5  
MOV R16, 62
LOAD R16, 5

_L1:

// Check for R0 < 8
MOV R16, 8
GT R16, R0
JZ R16, _L2

// R16 = 512 * 62 + R0
MOV R16, 62
MUL R16, 512
ADD R16, R0

// Transfer contents of memory location pointed to by R16 to R1
MOV R1, [R16]

// Printing word (R1)
PORT P1, R1
OUT

// counter = counter + 1
MOV R16, R0
ADD R16, 1
MOV R0, R16

JMP _L1
_L2:

HALT
