Device Handler Interrupt

Description: It handles interrupt from I/O devices

Change the device status register contents to 0.
Check the who issued the load/store instruction using Disk Status Table.
If the load/store instruction was issued by the timer interrupt, wake up all the processes in WAIT_MEM state.
If the load/store instruction was issued by a process, go through the Process Table and wake up those processes whose state is WAIT_DISK.
