eXpOS Implementation

Introduction



This document gives implementation of Data Structures and Algorithms used in eXpOS. It also provides the organizations of disk and memory in eXpOS.

Note 1 : The Inode table occupies the first 960 words (60 entries each of size 16 words) in the disk blocks 3 and 4. User table occupies the next 32 words (16 entries each of size 2 words) and the last 32 words are reserved for future use.

Note 2 : The Root File occupies the first 480 words of the 5th block and the last 32 words are unallocated.
