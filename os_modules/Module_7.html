Boot Module



This module is responsible for hand creating the INIT and SHELL processes. It loads the other modules, interrupt routines and disk data structures from disk to memory. It also initializes different memory data structures. Boot module is invoked by the OS Startup code at the time of booting.


Algorithm:

   Load exception handler, timer interrupt handler, disk interrupt handler,
   console (terminal) interrupt handler, library, all interrupt routines [4 - 17],
   modules [0 - 6], INIT, SHELL from disk to memory. 
   /* Module 7 (boot module) and IDLE will be loaded by OS startup code */
   See disk/memory organization.

   Load the disk data structures to the memory pages of their memory copies
   disk free list, root file, inode table, user table.

   /* Initialize the INIT (Login) process */

   Initialize the Page table base register to PAGE_TABLE_BASE + 20.
   Set the memory pages 63 and 64 for library in the page table. Set the reference bit to 0, valid bit to 1, write bit to 0.
   Set the heap page entries in the page table to -1 and valid bit to 0.
   Set the first code page entry to 65 (See memory organization). Set valid bit to 1 and write bit to 0. 
   Set remaining code page entries to -1 and valid bit to 0.
   Set the first stack page entry to 66 and valid bit to 1, write bit to 1. Set second stack page entry to -1 and valid bit to 0.
   
   Initialize the process table for INIT process.
   Initialize the Tick as 0, pid as 1, userid as 0, state as CREATED,
   user area page number as 77 (allocated from free user space), KPTR to 0, UPTR to 4096 (starting of first user stack page),
   PTBR to PAGE_TABLE_BASE + 20 and PTLR as 10.

   Store the IP value (from the header of the INIT) on top of first user stack page [66*512] = [65*512+1].
   
    /* Initialize the SHELL process */

   Initialize the Page table base register to PAGE_TABLE_BASE + 40.
   Set the memory pages 63 and 64 for library in the page table. Set the reference bit to 0, valid bit to 1, write bit to 0.
   Set the heap page entries in the page table to -1 and valid bit to 0.
   Set the first and second code page entries to 67 and 68 respectively (See memory organization). Set valid bit to 1 and write bit to 0.
   Set remaining code page entries to -1 and valid bit to 0.
   Set the first and second stack page entries to 78 and 79 (allocated from free user space). Valid bit to 1, write bit to 1.

   Initialize the process table for SHELL process.
   Initialize the Tick as 0, pid as 2, userid as 0, state as TERMINATED,
   user area page number as 80 (allocated from free user space), KPTR to 0, UPTR to 4096 (starting of first user stack page),
   PTBR to PAGE_TABLE_BASE + 40 and PTLR as 10.

   Store the IP value (from the header of the SHELL) on top of first user stack page [78*512] = [67*512 +1].

   /* Initialize all memory data structures */

   Set the states of all processes (other than INIT, IDLE and SHELL) in the process table to TERMINATED.
   Reset the status field in the terminal status table and disk status table to 0.
   Initialize the memory free list entries to 0 (free) or 1 (allocated).
   // presently 80 memory pages are allocated

   Initialize system status table entries. Mem free count as 47, wait mem count as 0, swapped count as 0, paging process as -1.
   Initialize disk map table for shell. First two code page entries with 9, 10 and all other entries to -1.
   Invalidate per process resource table of INIT and SHELL (all entries are set to -1).
   Store 0 in the process count field for all entries in Semaphore table.
   Initialize block number to -1, dirty bit to 0 and locking pid to -1 for the four buffer pages in the Buffer table.
   Set open file count and locking pid to -1 for all entries in the file status table. Set the inode index to -1 in the open file table for all entries.
Called by OS startup code.
