Resource Manager



This module is responsible for allocating and releasing the different resources. Note that the Terminal and Disk devices are freed by the corresponding interrupt handlers.

.



Function Number Function Name Arguments Return Value
1 Acquire Disk NIL NIL
2 Acquire Inode Inodeindex NIL
3 Acquire Terminal NIL NIL
4 Release Inode Inodeindex NIL
5 Close File File Descriptor, PID NIL
6 Acquire Buffer Buffer Number NIL
7 Release Buffer Buffer Number NIL

Acquire Disk

Description : Locks the disk device. 

Get the PID of the current process from the System Status Table.

while ( disk is locked ){        /* Check the Status field in the Disk Status Table. */
        Set state of the process as ( WAIT_DISK , - );
        Invoke the scheduler;
        }

Lock the disk by setting status field to 1 in the Disk Status Table.

return;
Called by DiskRead and DiskWrite functions.

Acquire Inode

Description : Locks the Inode entry corresponding to the inodeindex given as input.

Get the PID of the current process from the System Status Table.

while ( inode is locked ){   /* Check the Lock field in the File Status Table. */
        Set state of the process as ( WAIT_FILE , Inode Index );
        Invoke the scheduler;
    }

Lock the Inode by setting the Lock field in the File Status Table 
to the PID of the current process.;

return;
Called by Read,Write and Seek system calls.

Acquire Terminal

Description : Locks the Terminal device.

Get the PID of the current process from the System Status Table.

while ( Terminal device is locked ){    /* Check the Status field in the Terminal Status Table */
    Set state of the process as ( WAIT_TERMINAL , - );
    Invoke the scheduler;
    }
    
Lock the Terminal device by setting the Status field to 1 and the PID to the 
PID of the current process in the Terminal Status Table.

return;
Called by the Terminal Read and Termnal Write functions.

Release Inode

Description : Frees the lock of the inode entry corresponding to the inodeindex given as input.

Free the lock in the File Status Table corresponding to the inode index;       /* Set the Lock field to -1 */

loop through the process table{ 
       if (the process state is ( WAIT_INODE , Inode Index ) ){
             Set state of process as (READY , _ )
         } 
}
return;
Called by Read,Write and Seek system calls.

Close File

Description : Closes the file and frees the file descriptor.
Note: This function is not to be called if the file is locked.

Get the PID of the current process from the System Status Table.

Get the User Area  page number of the process from the Process Table entry
of the current process;

Get the Resource table entry corresponding to the file descriptor;

Get the Open File Table index form the Resource table entry;

Get the Inode index corresponding to the file from the 
Open File Table entry;

Decrement file open count in  File status table entry 
corresponding to the Inode index;

Decrement file open count in Open File Table entry;

if ( the file open count == 0){
    Invalidate the Open File Table entry;     /*  Set Inode Index field to -1*/
}

Invalidate the Resource Table entry;

return;
Called by the exit process function and close system call.

Acquire Buffer

Description : Acquire the buffer corresponding to buffer number given as input.

Get the PID of the current process from the System Status Table.

while ( Buffer is locked ){   /* Check the Locking PID field in the Buffer Status Table */
    Set state of the process as ( WAIT_BUFFER , Buffer Number );
    Invoke the scheduler;
    }

Lock the Buffer by setting the PID of the current process in the Locking PID field
in the Buffer Status Table ;

return;
Called by BufRead and BufWrite functions.

Release Buffer

Description : Release the buffer corresponding to buffer number given as input.

Free the lock in the the Buffer Status Table entry corresponding to
the buffer Number; /* Set Locking PID field to -1 */

loop through the process table{ 
       if (the process state is ( WAIT_BUFFER , Buffer Number ) ){
             Set state of process as (READY , _ )
         } 
}
return;
Called by BufRead and BufWrite functions.
