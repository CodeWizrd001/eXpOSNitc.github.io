Access Control Module



This module contains the functions that manages access locks.



Function Number Function Name Arguments Return Value
ACQUIRE_KERN_LOCK = 1 Acquire Kernel Lock NIL NIL
ACQUIRE_SCHED_LOCK = 2 Acquire Scheduler Lock NIL NIL
GLOCK = 3 Acquire Glock (Unused) NIL NIL
RELEASE_LOCK = 4 Release Lock LockVarAddress NIL


Acquire Kernel Lock

Description : Acquires the KERN_LOCK, which is a common access lock to be set before running any critical kernel code other than scheduling. This function must be called before performing any kernel function, this lock must be set by the kernel module/interrupt handler so that the other core waits till the critical action is completed.

    if (core is SECONDARY_CORE){
        if (PAGING_STATUS or LOGOUT_STATUS is on){
            Set the state of current process to READY

            Call the switch_context() function from the Scheduler Module.
        }
    }

    while ( tsl (KERN_LOCK) == 1 ){
        continue;
    }

    return;
Called by all system calls, exception handler and timer interrupt handler.


Acquire Scheduler Lock

Description : Acquires the Scheduler Lock, which is an access lock to run the Scheduler Module. This is called so that if one core has set the SCHED_LOCK in the Scheduler Module, the other core runs in a busy loop until execution of the Scheduler Module is completed. 

    while ( tsl (SCHED_LOCK) == 1 ){
        continue;
    }
    
    return;
Called by the Scehduler module.


Acquire Glock

Description : Acquires the General Purpose Lock, which is a general purpose lock variable that is left unused for future use.

    while ( tsl (GLOCK) == 1 ){
        continue;
    }
    
    return;



Release Lock

Description : Releases the access lock provided as the argument. 

    Store 0 at the address LockVarAddress.

    return;
Called by all system calls, exception handler, timer interrupt handler and scheduler.
