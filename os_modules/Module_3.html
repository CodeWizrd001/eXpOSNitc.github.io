File Buffer Manager



These routines are invoked from within system calls/interrupts/exception handler. Note that OS blocks only inside the modules. A module is designed to call only another module. It returns using return (RET machine) instruction.

The function number is passed by R1 and arguments are passed by registers R2, R3 etc in order. The return value (if any) is passed by R0.



Module_3 : File Buffer Manager


Function Number Function Name Arguments
1 Buffer Write Disk Number, Offset, Word
2 Buffer Read Disk Number, Offset, Memory Address

Buffer Write 


    Calculate buffer Number as Disk Number % 4;
    
    Acquire the buffer by calling the Acquire_Buffer() function
    in the Resource Manager module;
    
    if (the buffer contains a different disk block){
        
        if( the buffer contents are dirty){
              Write back the contents of the buffer
              to the disk by invoking disk_store() 
              function in the device manager module;
        }
        
        Load the required disk block into the buffer by invoking
        the disk_load() function in the device manager module;

        Set the new Disk block number in the Buffer table entry;
    }
    
    Write the contents of the word taken as input into the offset
    location in the buffer; 
    
    Mark the buffer as Dirty;
    
    Release the buffer by calling the Release_Buffer() function
    in the Resource Manager module;
    
    return;

Buffer Read 


    Calculate buffer Number as Disk Number % 4;
    
    Acquire the buffer by calling the Acquire_Buffer() function
    in the Resource Manager module;
    
    if (the buffer contains a different disk block){
        
        if( the buffer contents are dirty){
              Write back the contents of the buffer
              to the disk by invoking disk_store() 
              function in the device manager module;
              
              Mark the buffer as clean in the 
              corresponding buffer table entry;
        }
        
        Load the required disk block into the buffer by invoking
        the disk_load() function in the device manager module;

        Set the new Disk block number in the Buffer table entry;
    }
    
    Copy the contents in the offset location in the buffer to the
    physical address given as input; 
        
    Release the buffer by calling the Release_Buffer() function
    in the Resource Manager module;
    
    return;
