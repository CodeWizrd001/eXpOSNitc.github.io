Scheduler/Timer Interuppt

Description : Schedules the process

Argument : None

Return Value: None

1. Saving the context for current process
Store the context of the current process in its PT entry using Backup command.
2. Looking at other processes
Look at the System Status Table for Wait_mem_count and Mem_free_count. Based on these two, 4 cases can occur.
Mem_free_count > 0 Mem_free_count = 0
Wait_mem_count > 0 Case 1 Case 2
Wait_mem_count = 0 Case 3 Case 4
Case 1 : Scan through the process table and wake up all the processes in (Wait_Mem,_) state. Reset the Wait_mem_count to 0. Store the PID of the first (WAIT_MEM, _ ) process as Next_process. Go to Step 4.
Case 2 : Check whether disk is free. If yes, then do second chance algorithm.
  • Scan the valid entries in the Per-Process Page Tables of all processes in a circular manner and checks for a page with reference bit 0. The pages that are shared between processes, indicated by a number greater than 1 in its memory free list entry, is skipped in the search.
  • Whenever a page with reference bit 1 is encountered it is set to 0. When a page with reference bit 0 is encountered, it is selected as the victim page. The steps to swap out the victim page is given as follows:
    1. The Memory copy of Disk Free List is scanned for a free block in the swap area. If a free block is found, the victim page in memory is stored in this disk block.
    2. If the victim page is a stack page, change the state of that particular process to (SWAPPED,_). Increment the Swapped_count in System Status Table to 0.
    3. The Disk Free List is updated by incrementing the entry for the swap disk block as 1, indicating used. The entry for the victim page in the Per-Process Page Table is updated with the address of this block in the swap area of the disk and is set as invalid.
    4. The Memory Free list, Mem_free_count and Wait_mem count is updated. All the processes with (Wait_Mem, _ ) status are changed to (READY, _ ) state. Store the PID of the first (Wait_mem_count, _) process as Nest_process. Go to step 4.
If disk is not free, then go to Step 3.
Case 3: Check whether the disk is free. If yes, then look at Swapped_count. Go to Step 4.
