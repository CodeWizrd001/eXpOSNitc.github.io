Close System Call

Description: The Close system call is used to close a file.

Arguments: fileDescriptor

Return Value: 0 (Success)

1. Per-process Resource Table Validation
Check if fileDescriptor is valid by checking whether its value lies between 0 to XFS_MAXFDES. If invalid, return from the system call with error XFS_INVFDES, indicating failure. Check the entry corresponding to fileDescriptor is valid. If the entry is invalid, which is indicated by a -1, return from the system call with error code XFS_INVFDES. Othewise, store the File Table Index of that entry in FTindex.
2. Updation of the File Table
In the entry corresponding to FTindex, decrement the File Open Count. If the File Open Count becomes 0, invalidate the File Table entry.
3. Invalidate the Per-Process Resource Table entry.
4. Return from system call with 0 (success).
