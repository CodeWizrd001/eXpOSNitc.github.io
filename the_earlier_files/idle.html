Idle process

Arguments: None

Return Value: None

Description: It is a user program which is created and executed by the bootstrap loader. It is stored in the disk and will be loaded to memory by the bootstrap loader. The main purpose of the idle process is to run as a background process in an infinite loop. The page table and process table for the init process will be set up by the bootstrap loader. The PID of the init process is fixed to be 0.
